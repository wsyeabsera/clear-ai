## Quick Start

If you're eager to get started, here's the fastest way to run Clear AI:

```bash
# Clone and install
git clone <repository-url>
cd clear-ai
npm install

# Start all services
npm run dev
```

This will start:
- **Client**: http://localhost:3000
- **Server**: http://localhost:3001
- **API Docs**: http://localhost:3001/api-docs
